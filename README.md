# Elasticsearch Query DSL

A Ruby gem to build Elasticsearch queries.

## Installation

Add this line to your application's Gemfile:

    gem 'elasticsearch-query_dsl'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install elasticsearch-query_dsl
